Development

Most of the project's content (wiki, bug tracker, source repository) is available here (hosted by github).

The latest version of Erebot's code is available on github and may be cloned using the following command: ${'git clone git://github.com/fpoirotte/Erebot.git'} This will create a local repository with the core files required to run Erebot.
You may also be interested in checking out the code of other modules.
Last but not least, the online documentation contains additional information that you be useful to you, like a list of required dependencies for Erebot.

Buildbot

The project uses Buildbot to automatically generate a bunch of files from the latest version of the code.
You may follow the current state of every Erebot component on our Continuous Integration server
