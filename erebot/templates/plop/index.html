What is PLOP?

PLOP is a port of the Python logging module (which is itself a port of Java's log4j) on PHP.
The code is already featureful compared to other PHP logging systems and can be easily extended to fit one's specific needs.

Differences from the Python API

While most of the API is compatible between the Python logging module and PLOP, there are a few differences listed below.

  • Parameters for handlers in the configuration file are not yet implemented. [#260]
  • PLOP can also read its configuration from an XML file.
  • Not all handlers are supported by PLOP (especially the Windows Events Log handler).

Relation with Erebot

PLOP is used on Erebot to provide logging features. The XML configuration format is used.
